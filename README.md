# Train 

run python train.py in cmd with config

```
batch_size = 32
train_dir_path = "Path/To/DataSet"
data_shape = (4096, 4096)
method_dict ={
    "conv17":0,
    "inceptionresnetv2":1,
    "dense121":2,
    "efficientnet":3,
    "cmt":4,
    "ADFP":5,
    "PCT":6,
}

trainer = Train(
    image_shape = data_shape,
    class_number = 2, 
    is_show = False,
    name = "PCT",
    method_type = 6
)
```
